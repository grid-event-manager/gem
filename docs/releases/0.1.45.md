# GEM 0.1.45 Alpha

GEM 0.1.45 is an alpha release candidate for controlled testing.

## Highlights

- Added About page with product, copyright, license, and help link copy.
- Added Apache-2.0 license.
- Retained shared Compose UI across desktop and Android.
- Retained 23 in-app localization catalogues.
- Built Linux, Windows, macOS, and Android test artifacts from committed source.

## Artifacts

- Linux: `gema_0.1.45_amd64.deb`
- Windows: `gema-0.1.45.msi`
- macOS: `gema-1.0.45.dmg`
- Android: `gem-android-0.1.45-debug.apk`
- Checksums: `SHA256SUMS.txt`

## Install Notes

- Existing local account and preference state is intended to be preserved during normal over-the-top installs.
- Windows and macOS artifacts are not yet code-signed for public trust prompts.
- The Android artifact is debug-signed for sideload alpha testing.

## Known Limits

- GEM is alpha software.
- GEM is not affiliated with Linden Lab.
- GEM sends notices through the operator account's existing Second Life permissions and inventory.
- GEM does not diagnose every post-send grid delivery or archive confirmation condition in the UI.
- GEM Pro, serial keys, skin packs, and advanced event-management features are not part of this release.
