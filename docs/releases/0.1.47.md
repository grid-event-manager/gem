# GEM 0.1.47 Beta

GEM 0.1.47 is a beta release for controlled public testing.

## Highlights

- Provides the shared notice-sending workflow across Linux, Windows, macOS, and Android.
- Shows login, logout, attachment, and notice-sent confirmations through the shared operation modal.
- Keeps persistent attachment state in the inventory strip with the existing remove control.
- Retains shared Compose UI across desktop and Android.
- Retains 23 in-app localization catalogues.
- Ships release assets through GitHub Releases rather than committing binaries to the repository.

## Artifacts

- Linux: `gema_0.1.47_amd64.deb`
- Windows: `gema-0.1.47.msi`
- macOS: `gema-1.0.47.dmg`
- Android: `gem-android-0.1.47-debug.apk`
- Checksums: `SHA256SUMS.txt`

## Install Notes

- Existing local account and preference state is intended to be preserved during normal over-the-top installs.
- Windows and macOS artifacts are not yet code-signed for public trust prompts.
- The Android artifact is debug-signed for sideload beta testing.

## Known Limits

- GEM is beta software.
- GEM is not affiliated with Linden Lab.
- GEM sends notices through the operator account's existing Second Life permissions and inventory.
- GEM does not diagnose every post-send grid delivery or archive confirmation condition in the UI.
- GEM Pro, serial keys, skin packs, and advanced event-management features are not part of this release.
